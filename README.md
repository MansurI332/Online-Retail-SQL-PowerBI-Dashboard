# 📊 Online Retail Sales Dashboard (SQL + Power BI)

A data analytics project built using SQL and Power BI to analyse over 500,000 retail transactions and uncover insights into revenue performance, customer behaviour, and product sales.

The project covers the complete analytics workflow including data cleaning, exploratory analysis, customer segmentation (RFM), KPI development, and interactive dashboard design.

---

## 🚀 Overview

The objective of this project was to transform raw transactional data into a business intelligence solution that provides actionable insights for decision-makers.

Using SQL, the dataset was cleaned, analysed, and transformed into reporting views. These views were then connected to Power BI to create a multi-page dashboard focused on:

- Business performance monitoring
- Customer segmentation and value analysis
- Product performance evaluation

## 📂 Dataset

The project uses the Online Retail dataset containing transactional sales data from a UK-based online retailer between December 2010 and December 2011.

The dataset includes:

- Invoice Information
- Product Details
- Quantity Sold
- Unit Price
- Customer IDs
- Countries
- Transaction Dates

Over 500,000 transactions were analysed to identify business trends and customer purchasing behaviour.

---

## 🏗️ Project Workflow

### 1. Data Quality Assessment

- Identified missing customer IDs
- Removed invalid transactions
- Validated data consistency

### 2. SQL Analysis

Created reporting views for:

- Business KPIs
- Country Performance
- Monthly Revenue Trends
- Product Analysis
- RFM Segmentation

### 3. Customer Segmentation

Implemented an RFM framework to classify customers into:

- VIP Customers
- Loyal Customers
- Regular Customers
- Low Value Customers

### 4. Dashboard Development

Built a three-page Power BI dashboard focused on:

- Executive Reporting
- Customer Analytics
- Product Analytics

## 🔍 Key Insights

### Business Performance

- Generated £8.91M in total revenue
- Processed 18,536 orders
- Served 4,339 unique customers

### Customer Insights

- VIP customers represent only 8% of customers
- VIP customers generate significantly higher customer value
- Nearly 40% of customers fall into the Low Value segment

### Product Insights

- Paper Craft, Little Birdie generated the highest revenue (£168K)
- The same product recorded the highest sales volume (81K units)
- Revenue is concentrated among a relatively small number of products

### Geographic Insights

- The United Kingdom generated 82% of total revenue
- Netherlands was the largest international market
